# 📝 Todo List App

A clean and beginner-friendly **Todo List Web Application** built using core front-end web technologies. This project helps users organize daily tasks efficiently with a simple and responsive interface.

# Features

* ✅ Add new tasks
* ✅ Mark tasks as completed
* ✅ Delete tasks
* ✅ Clean and responsive UI
* ✅ Fast and lightweight

---

# Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
